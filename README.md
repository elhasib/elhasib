Most computer engineering education assumes you have access to expensive software licenses, high-end hardware, and stable internet. If you're learning in Cairo, Nairobi, or Lagos? Good luck. Commercial tools cost thousands of dollars. Textbooks are paywalled. Tutorials assume you're at MIT.

## üõ†Ô∏è **What We Build**

ELHASIB creates open-source educational tools that run on low-end hardware, work offline, and teach foundational computer engineering concepts‚Äîfrom transistors to quantum circuits.

### **Current Tools:**

#### üîå [**8-Bit CPU Simulator**](link)
Complete walkthrough of building a functional CPU from scratch in Logisim. Learn von Neumann architecture, instruction sets, and how computers *actually* work under the hood. No CS degree required.

**Why it matters:** Most students never see how a CPU functions at the logic gate level. This demystifies it.

#### üí° [**MachDesigner: Photonic Circuit Simulator**](link)
GUI-based tool for designing and simulating Mach-Zehnder Interferometers (MZI) with noisy analysis using Monte Carlo methods. Built in Python with Matplotlib.

**Why it matters:** Photonic computing is the future, but most educational tools are locked behind expensive licenses. This makes optical circuit design accessible to anyone with Python installed.

#### ‚öõÔ∏è [**Quantum Circuit Tutorials**](link)
Step-by-step guides for building quantum algorithms (Grover's search, Deutsch-Jozsa, etc.) using IBM's Qiskit. Includes worked examples, explanations of quantum gates, and performance analysis.

**Why it matters:** Quantum computing education assumes you already know quantum mechanics. We start from "what even is a qubit?"

#### üßÆ [**FEM Stress Analyzer**](link)
Open-source 2D Finite Element Method (FEM) software for stress analysis using plane strain approximation. Visualizes stress distribution with Von Mises approximation.

**Why it matters:** Commercial FEM software (ANSYS, COMSOL) costs tens of thousands of dollars. This does the fundamentals for free.

#### üìö [**Computational Physics Problem Sets**](link)
Curated collection of physics problems with full solutions‚Äînumerical methods, simulations, modeling. From pendulum dynamics to waveguide stress analysis.

**Why it matters:** Learning computational physics requires *doing* computational physics. These are the problem sets I wish I had when I started.

## üéØ **Our Mission**

**Decolonize computer systems education.**

For too long, cutting-edge computing knowledge has been concentrated in institutions that most of the world can't access. We're changing that by:

1. **Building tools that work offline** (because internet access is a privilege)
2. **Requiring zero licensing fees** (because education shouldn't be paywalled)
3. **Teaching foundational concepts** (not just surface-level tutorials)
4. **Making everything open-source** (so anyone can fork, modify, and improve)

If you're in Accra or Alexandria and want to learn quantum computing, you shouldn't need a Stanford library card. You should need curiosity and an internet connection long enough to download our tutorials.

That's the world we're building.
